温度效应
Softmax 温度缩放

温度参数控制注意力分布的尖锐程度。通过除以温度值来缩放得分矩阵：

softmax(Q @ K.T / sqrt(d_k) * T)
T<1 注意力更集中，分布更尖锐
T=1 标准 softmax，无缩放
T>1 注意力更分散，趋于均匀
注意力解读
如何解读热力图

每行代表一个 token 作为查询 (Query)，每列代表一个 token 作为键 (Key)。单元格颜色越亮，表示该 token 对应位置的注意力权重越高。

亮黄色区域：高注意力，token 间关联强
中间色调：中等注意力关联
深蓝/黑色区域：低注意力，几乎不关注

对角线通常最亮，表示自注意力。相邻 token 的注意力也往往较高，体现了局部依赖关系。

初始化方式对比
X
Xavier 当前

基于 Tanh/Sigmoid 设计，方差为 2/(fan_in + fan_out)。适合 Sigmoid 激活函数。

K
Kaiming (He)

基于 ReLU 设计，方差为 2/fan_in。更适合 Transformer 中常用的 ReLU/GELU 激活。

O
Orthogonal

正交初始化，保持矩阵正交性。有助于梯度稳定传播，RNN 和某些 Transformer 变体中表现优秀。
