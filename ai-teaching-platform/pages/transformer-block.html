位置编码可视化 正弦 / 余弦波图案
sin(pos / 10000^(2i/d_model))
cos(pos / 10000^(2i/d_model))
不同维度 i 对应不同频率的正弦/余弦波。低维度使用高频波，高维度使用低频波，使模型能区分不同相对位置的距离。
公式速查
Multi-Head Attention
MultiHead(Q, K, V)
= Concat(head_1, ..., head_h) WO
head_i = Attention(QWQ, KWK, VWV)
Attention(Q, K, V) = softmax(QKT / sqrt(d_k)) V
Feed-Forward Network
FFN(x) = max(0, xW1 + b1) W2 + b2
内部维度 d_ff 通常为 d_model 的 4 倍
W1: (d_model, d_ff)
W2: (d_ff, d_model)
Layer Normalization
LN(x) = gamma * (x - mu) / sigma + beta
mu = mean(x)，sigma = std(x)
gamma, beta 为可学习参数
在 d_model 维度上归一化
Positional Encoding
PE(pos, 2i) = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
pos = 位置索引
i = 维度索引
周期随维度递增
关键概念
残差连接
通过将输入直接加到子层输出上，解决深层网络中的梯度消失问题，使梯度可以直通传播。每个子层: output = LayerNorm(x + Sublayer(x))。
层归一化 (LayerNorm)
在特征维度上归一化，稳定每层的输入分布。与 BatchNorm 不同，LayerNorm 对每个样本独立计算，不依赖批量统计量。
多头注意力
将 d_model 拆分为 num_heads 个子空间，每个头独立学习不同的注意力模式。使模型能同时关注不同位置的不同表示子空间。
因果掩码 (Causal Mask)
Decoder 中使用的下三角矩阵掩码，将未来位置的注意力分数设为负无穷，确保自回归生成时只能看到已生成的 token。
缩放点积
Attention 计算中除以 sqrt(d_k)，防止高维点积值过大导致 softmax 梯度消失。当 d_k 较大时，点积的方差也随之增大。
